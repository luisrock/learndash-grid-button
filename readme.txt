**Grid Button Text for LearnDash** is the ultimate way to define custom texts and styles for courses buttons on the LearnDash Course Grid.

== Description ==
Grid Button Text for LearnDash** is the ultimate way to define custom texts and styles for courses buttons on the LearnDash Course Grid. You can set texts and styles based on course access mode (open, free, paynow, recurring, closed) mixed with user status (non-logged, non-enrolled, enrolled and completed the course).

Main features:

* define text based on course access mode
* define text based on user status related to each course
* define style based on user status related to each course
* choose to keep default text and style for some courses/users


== Installation ==
1. Upload plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Visit settings through "Grid Button Text" (LearnDash - LMS submenu) link.


== Frequently Asked Questions ==
= Does this plugin work without LearnDash? =

No, this plugin has been built specifically for LearnDash and requires LearnDash LMS AND LearnDash LMS - Course Grid to function. 

= Does this plugin work with any theme? =

That's the goal, but we cannot be sure. There are so many themes! If you have any troubles, please use the support forum
